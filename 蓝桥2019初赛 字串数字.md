## 题目来源：[蓝桥2019初赛 字串数字](http://oj.ecustacm.cn/problem.php?id=1476)

## 题意：小明用字母A对应数字1，B对应数字2……Z对应数字26，AA对应27，AZ对应25，LQ对应329，问LANQIAO对应的数字是多少？

### 思路：26进制的计算。

### AC代码：

```
#include<iostream>
#include<bits/stdc++.h>
using namespace std;
int main()
{
    long long int p=1;
    long long int ans=0;
    ans+=15;
    p*=26;
    ans+=p*1;
    p*=26;
    ans+=p*9;
    p*=26;
    ans+=p*17;
    p*=26;
    ans+=p*14;
    p*=26;
    ans+=p*1;
    p*=26;
    ans+=p*12;
    cout<<ans<<endl;
    return 0;
}

```
